# speed-cli

## Features

- Ping
- Bandwidth
- Jitter
- Download
- Upload
- MSS/MTU Size
- UDP
  - Bandwidth
  - Packet loss
  - [Delay jitter](https://en.wikipedia.org/wiki/Packet_delay_variation)

- Export data (CSV, JSON)
- Render report (HTML)
